A Game Boy emulator written in C to explore low-level systems programming and computer architecture. The project emulates core hardware components, including the CPU and memory.

My contributions:

  • Implemented CPU instruction decoding and register emulation in C
  • Refactored emulator subsystem interfaces to improve modularity and debugging
  • Authored architecture and debugging documentation for emulator components
