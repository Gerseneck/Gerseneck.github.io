An embedded chess clock built with Arduino, featuring customizable time controls, a responsive non-blocking architecture, and an intuitive LCD interface. Designed as an approachable introduction to embedded systems and firmware development.

Core features include:

  • Time Presets
  • Customizable Playtime
  • Advanced Logging
