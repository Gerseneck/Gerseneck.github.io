# Gerseneck.github.io

My personal website and blog post.

## Requirements

- Relatively recent version of Ruby
- Jekyll

## Setting it up locally

```
git clone https://github.com/Gerseneck/Gerseneck.github.io.git
cd Gerseneck.github.io/
bundle install
yarn install
```

## Running a local server

```
jekyll s
```

## Formatting

```
yarn prettier --write .
```
